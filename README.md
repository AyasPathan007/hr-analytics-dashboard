# 📌 Project Overview

This interactive **HR Analytics Dashboard** was developed in **Power BI** to help HR teams monitor workforce metrics, employee demographics, hiring trends, and employee attrition.

The dashboard transforms raw HR data into meaningful business insights, enabling faster and data-driven decision making.

---

# 🎯 Business Problem

Organizations often struggle to:

- Monitor employee attrition
- Track workforce demographics
- Analyze department-wise employee distribution
- Identify hiring trends
- Measure workforce performance
- Support strategic HR planning

This dashboard provides a centralized view of all major HR KPIs.
